# TriviaForge

[![CI](https://github.com/mxn2020/triviaforge/actions/workflows/ci.yml/badge.svg)](https://github.com/mxn2020/triviaforge/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Generates interactive trivia quizzes on any topic using AI

## Tech Stack
- **Frontend**: Vite + React + TypeScript
- **Styling**: Vanilla CSS (dark theme design system)
- **UI Library**: @geenius-ui/react-css
- **Backend**: Convex (Serverless)
- **Auth**: Convex Auth (email/password)
- **Payments**: Stripe (subscriptions)
- **AI**: NVIDIA NIMs
- **Analytics**: Vercel Analytics

## Quick Start

```bash
cd web
pnpm install
npx convex dev        # Start Convex backend
pnpm run dev          # Start Vite dev server
```
